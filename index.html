Waterpipe is software which helps to you write backend solution with low-code or no-code absolutely.

Introduction

This document describe API of waterpipe. You can create different versions of process, create custom nodes and start tasks.

Node types

waterpipe.start

Any process must start with this node.

Does not have arguments.

waterpipe.final

For complete status of task execution should arrive to final node.

Otherwise the task will stay in progress status

archive
boolean
Default: true

Set to false for disable storing task result

{
  • "archive": true
}

waterpipe.code

Execute any JavaScript code.

I advise you not to use it for simple set of parameter into task because V8 takes more time then builtin nodes.

script
required
string <base64>

Any JavaScript code. Should be in base64

important
boolean
Default: false

If set to true then agent will sync all logs of the task with master

{
  • "script": "Y29uc29sZS5sb2coImhlbGxvIHdvcmxkIik=",
  • "important": true
}

waterpipe.condition

"if-then" node. Compares two values and if got result true the task will go to the next node which specified in route argument

The compare works same as in JavaScript

key
required
string

jsonpath to task field

value
any

any value. Use $get(jsonpath) if you want to take value from another field of task

operation
required
string
Enum: ">" "<" ">=" "<=" "==" "==="

JavaScript condition operator

invert
boolean
Default: false

Invert compression value. (!false -> true)

route
required
string <UUIDv4>

Route to node if got true

{
  • "key": "foo.bar.value",
  • "value": "$get(one.two.three)",
  • "operation": ">",
  • "invert": false,
  • "route": "9e1de30f-2e61-46dd-b360-b5409effb9d3"
}

waterpipe.set

Set any value to field

path
string <jsonpath>

path to field

value
any

Any value. Use $get(jsonpath) if you want take value from another task

{
  • "path": "one.two.three",
  • "value": {
    • "foo": "bar"
    }
}

waterpipe.create_task

Create child task and send him to a process.

Use copy argument if you want take all fields from parent

copy
boolean or null
Default: false

Copy all fields from parent to child

required
Array of objects

Payload which need to set into child task

process
required
string <UUIDv4>

Process id

version
required
integer

Process version

node
required
string

Id of start node in target process

{
  • "copy": false,
  • "payload": [
    • {
      • "param": "one.two.three",
      • "value": "$get(some_fields.one)"
      }
    ],
  • "process": "4eb356fd-3c1b-49d1-95ca-26f4267439ad",
  • "version": 5,
  • "node": "a12846f8-09a2-45af-9f92-2d4110a12800"
}

waterpipe.goto

Jump current task to another process.

When target process finished the task return by stack to back and continue execution where his stopped.

process
required
string <UUIDv4>

Process id

version
required
integer

Process version

node
required
string

Id of start node in target process

{
  • "process": "4eb356fd-3c1b-49d1-95ca-26f4267439ad",
  • "version": 5,
  • "node": "a12846f8-09a2-45af-9f92-2d4110a12800"
}

waterpipe.api_call

Do HTTP request and set result to field. Supports only application/json

url
required
string

Destination by http

method
string
Default: "GET"

HTTP method

body
string

Request payload. Supports only application/json

set
string <jsonpath>

Field where should be stored result

{
  • "url": "http://example.com",
  • "method": "POST",
  • "body": "{\"hello\": \"world\"}",
  • "set": "response"
}

Process node model

id
required
string <UUIDv4>
type
required
string <package.name>

This is a qualifier of node. Describe for system what logic need execute

title
string

Show in editor this text on node element

args
object <key-value>

Node maybe require some arguments for his work

position
object

Used by editor for placing node in space

timeout
integer <seconds>

Timeout task in this node. If execution of task in this node will slow then after this time system will stop execution and set error "timeout"

next
string <UUIDv4>

Id of next node. When this node will be completed then execution go to node with this id
